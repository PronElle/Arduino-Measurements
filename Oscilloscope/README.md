* resolution: 10 bit
* 4 channels 
* analog -3 dB band:  ~ 1 MHz
* sampling frequency : 8 kSa/s
* equivalent sampling frequency : 16 MSa/s 
* refresh rate : 40 frame/s
* trigger : Edge
